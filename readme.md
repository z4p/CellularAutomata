# Анализ, синтез и применение клеточных автоматов (КА)

## Общие примечания
    * практическую часть выполнять на JS, рабочий вариант заливать на jsfiddle;
    * выделить 2 части для проработки: диплом для защиты (супрун-style), доклады для конференций (science-style);
    * супрун-style требования:
        1. цветное поле;
        1. ручное задание правил пересчета;
        1. анализ устойчивых состояний;

    * science-style задачи:
        1. формализация КА с использованием актуального матаппарата дискретной математики: найти готовую или написать (*теория информации*, *дискретка*);
        1. анализ применений КА:
            i. в распределенных и параллельных вычислениях;
            i. в обработке информации:
                - сжатие информации с потерями;
                - восстановление информации;
                - удаление шумов и дефектов;
                - криптография: необратимое хеширование;
                - генерация псевдослучайных чисел;
            i. в задачах моделирования физических процессов;
        1. физическая реализация (на ПЛИС, квантовых операциях, ДНК-операциях).
    
## Цель студента
Выбрать пробел в общемировой копилке знаний и заполнить его максимально качественно и эффективно.
